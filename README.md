# Photonic Device Reliability – Machine Learning Framework

## Overview
This repository provides a complete machine learning framework for analyzing and predicting the reliability of photonic devices, including laser diodes and InP-based integrated circuits.

It includes:
- A structured dataset (synthetic example provided)
- Feature engineering strategies based on physical parameters
- Multiple machine learning models (tree-based, neural networks, linear models)
- Automated training, tuning, and evaluation pipeline
- Visualization and reporting tools

This work is designed to support research in **device degradation, failure prediction, and reliability modeling**.
